Final Countdown jQuery Plugin Demo

This plugin can be used to create a real-time accurate countdown clock.

This plugin grabs the current time from an API call to avoid incorrect time settings on the user's device. The format of the clock is customisable as is a message that is displayed with the clock reaches zero. Here is the clock in action :

 

You can initialize the plugin a few different ways.

Quick and dirty method:

Simply specify an end time (mandatory) in ISO8601 format :

$.fn.countdown({ 'end_time': '2025-11-29T05:00:00-05:00' });

Custom Options:

allows any number of optional customizations :

$.fn.countdown({ 'end_time': '2025-11-29T05:00:00-05:00', 'counter_selector': '#countdown', 'intro_message': 'The world will <strong>END</strong> in ', // can include HTML 'complete_message': 'The world has ended!', // can include HTML 'days_label': 'days,', 'hours_label': 'hours,', 'minutes_label': 'minutes,', 'seconds_label': 'seconds.', 'time_api': 'http://www.timeapi.org/utc/now.json' });

This plugin has some potential conflicts. For browsers using IE8 and lower, it adds a method to the JS Date object that adds the ability to parse dates in ISO8601 format. If you have another script that also does this, you may see unexpected results from one of the scripts. Also, the call method decribed above assumes jQuery is using it's default namespace.
